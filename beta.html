More Detailed Instructions!

You'll receive an email to install the "Test Flight" app (Apple's beta testing software). Install the Test Flight app and follow the instructions. It will install the Crossword Countdown app on your phone. Make sure to enable notifications in Test Flight so you can be notified of updates to the app as I release them.

I'm looking for feedback in two groups:

  1. Something is broken: A page doesn't load properly, the app crashes, a button is partly off screen, text is cutoff, etc.
  2. Suggestions to make it better: Is something confusing? Would you alter buttons? Are the puzzles too hard or easy? Etc.

You can send any feedback to me at dan.drabik@gmail.com or through the Test Flight app (which includes helpful device information).

Thanks, and hope you like it!
